在上一篇文章(Snackbar简单封装)中提到了BaseActivity的实现，其中包含一个ToolbarWrapper。

Toolbar是安卓中常见的一个控件，在我的项目中几乎所有Activity都会用到Toolbar。如果在每个Xml文件中都写一次Toolbar代码，将会写下许多重复代码。一旦某一天想把Toolbar的布局进行一些调整，就要调整所有Activity，这样显然是不优雅的。所以，我对Toolbar进行了封装。

最初我把相关代码直接写在BaseActivity中，不过发现随着需求的增多，代码量也变得更多了。所以我就把它单独拿出来作为一个类来使用，起名为ToolbarWrapper。

完整代码：https://github.com/Ethan-yt/MyClub/blob/master/app/src/main/java/com/ethan/myclub/main/ToolbarWrapper.java

Toolbar默认布局：https://github.com/Ethan-yt/MyClub/blob/master/app/src/main/res/layout/view_toolbar.xml

功能说明

  • setTitle(String title) 设置标题 默认左对齐
  • transparent() 背景全透明
  • target(ViewGroup target) 设定Toolbra的父布局 若不设定默认为根布局
  • setTitle(String title, boolean isCenter) 设定居中的标题
  • withAnimate() 载入Toolbar时附带淡入动画
  • setScrollable() 支持某些Behavior自动隐藏 也就是设定SCROLL和ENTER_ALWAYS
  • setToolbarFitsSystemWindows() 设定toolbar的padding，留出状态栏的高度
  • setMenu(@MenuRes int resId, Toolbar.OnMenuItemClickListener onMenuItemClickListener) 显示菜单
  • setMenu(@MenuRes int resId, Toolbar.OnMenuItemClickListener onMenuItemClickListener, BaseActivity.OnFinishCreateMenu onFinishCreateMenu) 显示菜单并设置回调函数
  • showBackIcon() 显示左上角返回按钮
  • showNavIcon(@DrawableRes int navIconId, View.OnClickListener onClickListener) 自定义左上角按钮
  • setTextColor(int color) 设定文字颜色
  • show(@LayoutRes int layoutResId) 加载自定义布局的Toolbar
