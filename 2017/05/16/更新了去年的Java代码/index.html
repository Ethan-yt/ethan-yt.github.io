最近听一些学弟说，Java课程布置了大作业，想起去年这时，我也修了这门Java课程，时隔一年，作业的要求还是差不多的。去年我就在Github上开源了我的代码，只不过没有做过宣传，几乎没人看过。今年我稍作修改，发布一下，希望给学弟学妹一些参考，同时如果有改进的地方欢迎提交Pull Request。

Project的格式是IntelliJ IDEA，它是一款优秀的IDE，希望真正想学Java的同学卸载掉自己的Eclipse，使用这款IDE。编码采用UTF8。

如果还是想用Eclipse，并且在项目格式/编码遇到问题，请自行搜索 IntelliJ Eclipse 工程转换。

两个项目内均有文档，word格式，解释了涉及的算法等概念。
